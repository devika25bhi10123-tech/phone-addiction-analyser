# phone addiction analyser

## Description

The Phone Addiction Analyser is a Python-based command-line application designed to help users monitor and analyze their daily screen time. The project addresses the growing issue of excessive phone usage by allowing users to input their daily usage, store the data, and generate meaningful insights such as average screen time, usage trends, and addiction levels. Based on predefined thresholds, the system classifies users into categories like Healthy, Moderate, or Addicted and provides personalized suggestions to improve digital habits. This project demonstrates the practical application of programming concepts such as file handling, conditional logic, user input validation, and data analysis.

## Features

- Screen time tracking
- Addiction level classification (Healthy / Moderate / Addicted)
- Data storage using file handling
- Usage report with:
  -Average screen time
  -Highest and lowest usage
  -Total days tracked
- Smart suggestions based on usage
- User-friendly interface with clean output and messages
